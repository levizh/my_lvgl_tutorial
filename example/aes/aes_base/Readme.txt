功能描述
================================================================================
本样例主要展示AES加密，解密功能，在程序中写入一组需要加密的数据和密钥，串口打印
加密或者解密后的数据



================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_HC32F4A0_LQPF176_050_V10

辅助工具:
---------------------
无

辅助软件:
---------------------
串口调试助手

================================================================================
使用步骤
================================================================================
1） 打开工程编译并全速运行。
2） 串口调试助手查看加密和解密后的数据。

================================================================================
注意
================================================================================
密钥的长度只能为128位，192位，或者256位。
