功能描述
================================================================================
本样例主要展示TIMER2基本定时器功能，实现了用TIMER2定时1毫秒。

说明：例程中，TIMER2的时钟源为同步时钟源HCLK。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_HC32F4A0_LQPF176_050_V10

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译，将程序下载至目标板；
2）将示波器探头连接至P30，会看到示波器显示500Hz的方波。
