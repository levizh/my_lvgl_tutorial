功能描述
================================================================================
本样例为CMP外设配置为普通比较模式时,选择DA1O1为参考电压（负端电压），PGA1为比较
电压（正端电压），通过中断输出比较结果的样例。

测试环境
================================================================================
测试用板:
---------------------
EV_HC32F4A0_LQPF176_050_V10

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译。
2）芯片引脚P13连接输入比较电压（<VCC）。
3）全速运行样例代码。
4）调节输入比较电压值，查看LED灯变化。
5）当比较电压>参考电压，蓝色LED亮；
   当比较电压<参考电压，蓝色LED灭。
