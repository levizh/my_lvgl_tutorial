功能描述
================================================================================
本样例为CMP外设配置为普通比较模式时,选择CMP1_INM3为参考电压（负端电压），PGA2为
比较电压（正端电压），通过中断输出比较结果的样例。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_HC32F4A0_LQPF176_050_V10

辅助工具:
---------------------
稳压电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译;
2）芯片引脚PA7连接输入参考电压(<VCC),PA1连接稳压电源正端(<VCC);
3）全速运行样例代码;
4）调节稳压电源输入比较电压值，查看LED灯变化;
5）当比较电压<参考电压，蓝色LED亮;
   当比较电压>参考电压，红色LED亮;


================================================================================
注意
================================================================================
稳压电源的输入电压不要超过VCC
