平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示了CMP窗口比较模式，通过CMP1和CMP2两个通道实现，配置如下：
DAC1_OUT2为窗口高电压，本样例约为1.8V；
DAC1_OUT1为窗口低电压，本样例约为1.3V；
CMP1_INP3(PE10)和CMP2_INP2(PGA2)(PA01)为输入比较电压,需要外部连接；
通过VCOUT2引脚输出比较结果。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）芯片引脚PE10和PA01连接，稳压电源正端输入比较电压（<VCC）；
3）芯片引脚VCOUT2(PB13)连接示波器；
4）全速运行样例代码；
5）调节输入比较电压值，通过示波器查看引脚VCOUT2(PB13)变化。
   当比较电压<1.3V，VCOUT2输出低，LED_B熄灭；
   当比较电压>1.8V，VCOUT2输出低，LED_B熄灭；
   当1.3V<比较电压<1.8V，VCOUT2输出高，LED_B点亮；


================================================================================
注意
================================================================================

若选择PGAx作为比较电压源，需要使能PGA功能；
