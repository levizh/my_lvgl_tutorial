功能描述
================================================================================
本工程为CMP外设通过M0P_CMP1和M0P_CMP2两个通道实现窗口比较模式的样例，配置如下：
芯片内部Vref（具体电压请参考芯片数据手册）为窗口高电压；
IVREF1为窗口低电压；
IVCMP2_0为输入比较电压；
通过VCOUT引脚输出比较结果。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译。
2）芯片引脚IVCMP2_0（P23）连接输入比较电压（<VCC）。
3）芯片引脚IVREF1（P12）输入窗口低电压（0.8V）
4）全速运行样例代码。
5）调节输入比较电压值，通过示波器查看引脚VCOUT2（P61）变化。
   当比较电压<0.8V，VCOUT2输出低；
   当比较电压>Vref，VCOUT2输出低；
   其他电压VCOUT2输出高。
