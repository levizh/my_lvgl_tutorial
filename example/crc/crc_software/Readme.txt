功能描述
================================================================================
本例程实现了CRC16和CRC32的硬件计算和软件计算。while(1)中对CRC硬件计算和软件计算
的结果进行对比，一致则绿灯亮，否则，红灯亮


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_HC32F4A0_LQPF176_050_V10

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）用户可根据自己的需求，设置CRC初始值；
2）启动IDE的调试功能，断点查看程序执行情况；
3）观察LED_G灯常亮，计算结果一致；


================================================================================
注意
================================================================================
CRC_Calculate 函数的最后一个参数表示输入数据的位宽，请严格按照输入数据的
位宽设置该参数。
