功能描述
================================================================================
本样例为EFM扇区擦除，单编程，单编程回读，编程正确亮蓝灯，不正确亮红灯。
本样例以扇区10为例。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_HC32F4A0_LQPF176_050_V10

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，138行，142行，149行打上断点；
3）打开memory窗口，运行程序，可以看到地址0x00014000进行了擦除，编程操作；
3）蓝灯亮，编程正确。
