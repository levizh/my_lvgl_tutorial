功能描述
================================================================================
本样例展示EFM窗口保护功能，保护区域内能正确擦写，亮蓝灯；按下按键后，在保护区域
外擦写，触发中断（编程擦写错误中断），亮红灯。

---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行；
3）在保护区域内正确擦写，蓝灯亮；
4）按下按键，在保护区域外擦写，产生编程错误中断，红灯亮。
