功能描述
================================================================================
本样例为EFM一次性可编程字节（OTP）样例。样例中对OTP sector15(0x0001E000——0x0001FFFF)
先进行擦除、编程，然后锁定该区域，通过对该区域再次进行编程判断锁定是否成功。
若锁定成功，则第二次编程失败，绿灯灭，红灯亮。

================================================================================

---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行；
3）绿灯亮，先按下SW1键，再按下SW2键，绿灯灭，红灯亮（表示锁定成功）。


================================================================================
注意
================================================================================
OTP区域一旦锁定，则无法进行解锁，该区域将永久锁定。
进行OTP锁定时务必对照用户手册正确填写锁存区域地址。
