平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示ETH的TCP Server通信功能。

说明：
本样例配置ETH工作在TCP Server模式，配置TCP&UDP测试工具为TCP客户端，目标板LED_B亮
起后，单击TCP&UDP测试工具的"连接"按钮连接TCP服务器，连接成功后，通过TCP&UDP测试
工具发送一串数据，目标板收到数据后将数据回发给TCP客户端。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F4A0_LQ176_V10

辅助工具:
---------------------
网线、PC

辅助软件:
---------------------
TCP&UDP测试工具

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，关闭IDE下载界面；
3）将网线连接到目标板和PC的网络端口，设置PC的对应网卡的IP地址信息，如下：
    IP address：192.168.1.120
    Subnet mask: 255.255.255.0
    Default gateway: 192.168.1.1
    Preferred DNS server: 192.168.1.1
4）打开TCP&UDP测试工具，配置为TCP客户端模式，目标地址：192.168.1.20，
目标端口：5000；
5）等待目标板LED_B亮起，点击测试工具的"连接"按钮，触发连接TCP Server功能，连接成
功后可以在测试工具上看到图标状态变为绿色；
6）在窗口的发送区输入一串字符，点击“发送”，在接收区看到发送的字符串，则通信正常。
