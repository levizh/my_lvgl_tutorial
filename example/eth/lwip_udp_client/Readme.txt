平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示ETH的UDP Client通信功能。

说明：
本样例配置ETH工作在UDP Client模式，配置TCP&UDP测试工具为UDP服务器，目标板LED_B亮
起后，单击按键SW10连接UDP服务器，连接成功后，通过TCP&UDP测试工具发送一串数据，目
标板收到数据后将数据回发给UDP服务器。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F4A0_LQ176_V10

辅助工具:
---------------------
网线、PC

辅助软件:
---------------------
TCP&UDP测试工具

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，关闭IDE下载界面；
3）将网线连接到目标板和PC的网络端口，设置PC的对应网卡的IP地址信息，如下：
    IP address：192.168.1.120
    Subnet mask: 255.255.255.0
    Default gateway: 192.168.1.1
    Preferred DNS server: 192.168.1.1
4）打开TCP&UDP测试工具，配置为UDP服务器模式，本地地址：192.168.1.120，
本地端口：3000；
5）等待目标板LED_B亮起，点击目标板SW10按键，触发UDP Client连接功能，连接成功后可
以在UDP服务器看到客户端的IP地址为：192.168.1.20，端口为随机分配值，及一串字符数据；
6）在窗口的发送区输入一串字符，点击“发送”，在接收区看到发送的字符串，则通信正常；
7）再次点击目标板SW10，主动断开UDP连接。
