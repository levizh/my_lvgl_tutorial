平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示INTC模块EXINT检测功能。

说明：
本样例通过开发板上的KEY1，KEY4，KEY7引脚中断功能。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_HC32F4A0_LQPF176_050_V10

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序并运行；
3）按下KEY1（EXINT0）-->LED_BLUE交替亮灭；
4）按下KEY4（EXINT3）-->LED_RED交替亮灭；
4）按下KEY7（EXINT7）-->LED_RED和LED_BLUE交替亮灭；

================================================================================
注意
================================================================================
1. KEY1使用全局中断入口No.0~31中的No.0；
2. KEY4使用分组中断入口No.32~39中的No.33;
3. KEY7使用共享中断入口No.128
