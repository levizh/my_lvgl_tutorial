功能描述
================================================================================
本样例展示了SHA-256模式下，HASH使用DMA搬运数据并进行计算,计算结果与HASH计算器
计算结果比较，一致则输出。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_HC32F4A0_LQPF176_050_V10

辅助工具:
---------------------
无

辅助软件:
---------------------
串口调试助手

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行；
3）串口助手查看HASH计算结果；


================================================================================
注意
================================================================================
若需要计算的数据格式是字符串，需要选择main.c 81行数据格式为字符串。
