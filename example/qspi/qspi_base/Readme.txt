功能描述
================================================================================
本样例展示GPIO模块Pin的输出功能。

说明：
本样例通过GPIO引脚输出高低电平，驱动对应LED亮灭。

================================================================================

================================================================================

---------------------

---------------------
无

---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序并运行；
3）观察开发板上的LED（D26）亮灭。
