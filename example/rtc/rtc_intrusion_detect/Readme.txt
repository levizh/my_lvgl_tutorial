平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示RTC入侵检测功能。

说明：
本样例设置RTC的时间格式为24小时制，初始时间为20年10月10日23点59分55秒，再设置入
侵检测引脚为PI8，下降沿触发入侵事件，触发入侵事件后将触发事件保存到时间戳中，并
通过串口输出触发的端口及触发事件信息。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_HC32F4A0_LQPF176_050_V10

辅助工具:
---------------------
Micro USB数据线

辅助软件:
---------------------
串口调试助手

================================================================================
使用步骤
================================================================================
1）使用USB线连接电脑及目标板；
2）在串口调试助手SecureCRT上打开USB对应串口，配置端口如下参数：
     波特率：115200
     数据位：8
     校验位：None
     停止位：1
3）打开工程并重新编译，启动IDE的下载程序功能，关闭IDE下载界面；
4）观察串口打印的时间信息；
5）拉低PI8引脚触发入侵事件，观察到串口输出入侵端口及入侵时间信息。
