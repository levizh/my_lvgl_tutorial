功能描述
================================================================================
本样例展示TRNG产生64位随机数，在TRNG完成中断中通过串口打印。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_HC32F4A0_LQPF176_050_V10

辅助工具:
---------------------
无

辅助软件:
---------------------
串口调试助手

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行；
3）串口调试助手观察生成的随机数；


================================================================================
注意
================================================================================
为了生成更好的随机数，请将PCLK时钟频率设置为1MHZ以下，移位数设置成64位及以上。
